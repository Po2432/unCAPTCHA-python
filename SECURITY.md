üëâ [Submit a report here](https://github.com/Po2432/unCAPTCHA-python/security/advisories/new)

### What to include
- A clear description of the vulnerability  
- Steps to reproduce the issue  
- Affected version(s)  
- Any proof-of-concept code or screenshots (if available)

### What to expect
- We will acknowledge your report within **3 days**.  
- You will receive updates at least once every **10 days** until resolution.  
- If accepted, we will work on a fix and publish a security advisory with details.  
- If declined, we will explain why (e.g., not reproducible, expected behavior).  

### Disclosure
Please allow us up to **95 days** to address the issue before public disclosure. Coordinated disclosure helps protect users while we prepare a safe release.
